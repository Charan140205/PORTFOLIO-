# 💼 Personal Portfolio Website

A modern and responsive personal portfolio website developed using **HTML5**, **CSS3**, and **JavaScript**. This portfolio showcases my skills, education, projects, certifications, and professional experience with a clean and interactive user interface.

---

## 📌 Overview

This portfolio website serves as my professional online profile, highlighting my technical skills, academic background, projects, certifications, and contact information. It is fully responsive and designed with a modern dark-themed interface for an engaging user experience.

---

## 🚀 Features

- Responsive Design
- Interactive Navigation
- Hero Section with Profile
- Technical Skills
- Professional Experience
- Projects Showcase
- Education & Certifications
- Contact Form
- Resume Download
- Social Media Links

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Charan140205/Portfolio.git
```

2. Open the project folder

```bash
cd Portfolio
```

3. Open **index.html** in your browser.
